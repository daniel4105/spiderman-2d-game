# 🕷️ Spider-Man: Miles Morales 2D

Juego 2D desarrollado con **HTML, CSS y JavaScript usando Canvas API**, donde controlas a Miles Morales saltando entre edificios, lanzando telarañas y derrotando enemigos.

---

## 🎮 Características

* Movimiento lateral con físicas (gravedad y salto)
* Sistema de combate con telarañas
* Enemigos con IA básica y disparos
* Sistema de vidas ❤️
* Niveles progresivos con dificultad creciente
* HUD en tiempo real (vida, puntos, nivel, récord)
* Guardado automático de récord (LocalStorage)
* Pantallas de pausa, game over y victoria

---

## 🕹️ Controles

| Acción            | Tecla   |
| ----------------- | ------- |
| Moverse           | ← →     |
| Saltar            | Espacio |
| Disparar telaraña | Click   |
| Pausa             | ESC     |

---

## 🧠 Mecánicas

* Cada edificio contiene enemigos
* Ganas puntos al derrotarlos
* La dificultad aumenta por nivel:

  * Más enemigos
  * Más vida
  * Mayor velocidad de disparo
* Tienes un máximo de **5 vidas**

---

## 🛠️ Tecnologías utilizadas

* HTML5

* JavaScript (ES6)
* Canvas API

## 🚀 Conclusión

Este proyecto demuestra cómo se puede crear un videojuego funcional usando únicamente tecnologías web.
El uso de Canvas, junto con lógica en JavaScript, permite simular físicas, animaciones e interacción en tiempo real, haciendo el desarrollo más dinámico y creativo.

Además, el uso de IA como apoyo en el desarrollo facilita la resolución de problemas y acelera la construcción del proyecto.
